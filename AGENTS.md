# Project Rules

- This is a one-page academic-style blog site written for beginners.
- Visual style: white background, royal blue headings/titles, black body text, and light blue accents.
- Use CSS variables for colors, spacing, and radius.
- Use semantic HTML and accessible markup.
- Keep layout responsive for mobile and desktop.
- Keep the article in a centered readable container.
- Do not add frameworks, backend features, analytics, forms, or third-party UI libraries.
- Prefer simple, maintainable code over clever code.
- Keep `index.html` at the top level of the repo for Cloudflare Pages.
- The final site should feel polished, calm, and professional.
